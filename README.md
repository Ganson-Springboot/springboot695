## 4.1  前台功能模块
学生就业管理系统，学生、企业进入到网站首页，可以查看首页，企业招聘，求职信息，就业知识，个人中心，后台管理等内容进行操作，如图4-1所示。

图4-2学生注册界面图

图4-3企业招聘界面图

图4-4求职信息界面图
